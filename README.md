# Calculadora Básica em JavaScript

Calculadora simples feita em HTML, CSS e JavaScript puro.

## Funcionalidades
- Operações básicas (+, -, *, /)
- Parênteses com validação
- Porcentagem
- Raiz quadrada
- Quadrado
- Inverso (1/x)
- Botão de apagar (backspace)

## Tratamento de erros

- Divisão por zero

- Expressões inválidas

- Resultados infinitos (Infinity)

- Resultados inválidos (NaN)

Quando ocorre erro, o display é limpo automaticamente ao inserir novos valores.

## Regras de funcionamento

- Não permite operadores inválidos em sequência

- Não permite fechar parênteses sem abertura

- Não permite iniciar a expressão com operador

- Mensagem de erro não se concatena com novos números

- Expressões só são avaliadas quando válidas

## Tecnologias
- HTML
- CSS
- JavaScript (vanilla)

## Como usar:

git clone https://github.com/Prompts085/Calculadora-B-sica.git

*Abra o arquivo index.html diretamente no navegador.*

- Não é necessário servidor
- Funciona em qualquer navegador moderno

## Objetivo
Este projeto foi desenvolvido com o objetivo de:

- Praticar lógica de programação

- Entender o funcionamento interno de uma calculadora

- Trabalhar validações manuais de entrada

- Exercitar tratamento de erros em JavaScript

- Consolidar conhecimentos sem uso de frameworks
