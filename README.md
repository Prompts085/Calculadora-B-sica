# Calculadora Básica em JavaScript

Calculadora simples feita em HTML, CSS e JavaScript puro.

## Funcionalidades
- Operações básicas (+, -, *, /)
- Parênteses com validação
- Porcentagem
- Raiz quadrada
- Quadrado
- Inverso (1/x)
- Botão de apagar (backspace)
- Tratamento de erros (divisão por zero, expressões inválidas)

## Tecnologias
- HTML
- CSS
- JavaScript (vanilla)

## Objetivo
Projeto criado para praticar lógica, manipulação do DOM e controle de erros em JavaScript.
